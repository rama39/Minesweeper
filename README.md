# Minesweeper

Command line minesweeper in C

```
   +--+--+--+--+--+--+--+--+
   | 0| 1| 2| 3| 4| 5| 6| 7|
+--+--+--+--+--+--+--+--+--+
| 0|  |  |  |  |1 |1 |2 |# |
| 1|  |  |  |  |1 |# |2 |1 |
| 2|1 |1 |1 |  |1 |2 |3 |2 |
| 3|2 |# |1 |  |  |1 |# |# |
| 4|# |2 |2 |1 |1 |1 |2 |2 |
| 5|1 |1 |1 |# |1 |  |  |  |
| 6|  |  |1 |2 |2 |1 |1 |1 |
| 7|  |  |  |1 |# |1 |1 |# |
+--+--+--+--+--+--+--+--+--+
```

# Usage

./minesweeper
    Width: [board width]
    Height: [board height]

./minesweeper [board width] [board height]

Obs: minimun board size = 8
